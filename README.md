# 42_seoul_minishell
간단한 shell 만들기

함수    : 단어_단어
파일    : 명령어(ft), 나머지는 꼴리는대로
변수    : 놈규칙 쓰기




# 프로젝트 경로

```
includes
// 헤더파일
    minishell.h : 모든 헤더파일들을 가지고 있음
    command.h
    parsing.h
    utils.h
lib
    libft

srcs
// 소스파일
    command
        // 명령어 관련파일
    parsing
        // 명령어 입력부분
    error
        // 에러처리
    utils
        // 다른 파일들에 자주 사용되는 함수
    main.c : 시작 부분
```
